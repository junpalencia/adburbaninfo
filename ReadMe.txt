Urban Information System is a satellite-based Urban Development Information System where changes in historical data are being shown in map and charts since 1990. There are selected cities represented in points that are scattered all over ASIA.

Development Libraries:

- GeoExt 2
- OpenLayers 2
- Ext 4


Data:

The vector data are stored in PostGIS Database while raster data in GeoTIFF format.

Source code:

index.html - main cointainer where javascripts are loaded.
loader.js javascript file where paths to GeoExt 2 and Ext 4 are declared. 
map.js - javascript for main map. 
city.html - container for city.js.
city.js - main code for showing detailed city map.
